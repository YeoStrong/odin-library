# Odin-Library (The Odin Project)

A simple library application built with Vanilla JavaScript, focusing on Object Constructors and Prototypes.

## [Live Demo](https://yeostrong.github.io/odin-library/)

### Key Features

- **Book Management**: Add, delete, and toggle read status of books.
- **Persistence**: Data is saved to `localStorage`, so your library stays even after a refresh.
- **OOP Principles**: Used JavaScript prototypes to manage object methods.

### Tech Stack

- HTML5 / CSS3
- JavaScript (Vanilla JS)
- LocalStorage API
